| X();          |  | 创建一个匿名空容器 | 固定 |

| X u = a;      |  | 作用同于 X u(a) | 固定 |
| r = a;        | X& | 调用复制运算符后r==a | 固定 |

| a.begin()     | 迭代器 | 返回指向容器第一个元素的迭代器 | 固定 |
| a.end()       | 迭代器 | 返回超尾值迭代器 | 固定 |
| a.size()      | 无符号整型 | 返回元素个数，等价于a.end()-a.begin() | 固定 |
| a.swap(b)     | void | 交换a和b的内容 | 固定 |

| a != b        | 可转换为bool | 返回!(a--b) | 线性 |
